# Documentation

Welcome to the Documentation of Easy-dApp.
It contains seperated Documentation for all the Modules.
Get started with the Core Module or learn how to integrate everything with your Framework of choice. Currently we are supporting React, Next, Svelte and Vue.

### Quickstart

For a Quickstart you can utilize our CLI tool, to setup a Full Stack repository with the Framework of your choice.

```npx easy-dapp init```

After running the command you can provide all your details to customize the setup.
